用hexo+github搭载个人博客

发表于 2019-12-17

搭载步骤

1.下载安装git
2.下载并安装Node.js
3.安装Hexo
4.GitHub创建个人仓库
5.生成SSH添加到GitHub
6.将hexo部署到GitHub

下载安装Git

Git下载

点击后直接下载安装到自己想装的位置，如何打开安装，安装过程中一直点击next即可。安装完成后桌面会生成一个Git Bash图标，点击进去后（也可以用cmd）输入下面指令。

git --version

查看当前的git版本。

下载安装Node.js

Node.js下载

选择对应的版本下载到自己指定的位置即可，如何打开安装，过程也是一直点击next， 不过在Custom Setup这一步记得选 Add to PATH ,这样你就不用自己去配置电脑上环境变量了，装完在按 win + r 快捷键调出运行，然后输入cmd确定，在cmd中输入path可以看到你的node是否配置在里面（环境变量） ，正常来说都会有。

若安装成功后可查看node版本

GitHub创建个人仓库

若你还没有自己的GitHub账号的话，就要去GitHub官网上注册一个账号。注册完登录后，在GitHub.cogm中看到一个New repository，新建仓库时创建一个和你用户名相同的仓库，yourusername.github.io，只有这样，将来要部署到GitHub page的时候，才会被识别，也就是xxxx.github.io，其中xxx就是你注册GitHub的用户名。我这里是已经建过了。

ssh，简单来讲，就是一个秘钥，其中，id_rsa是你这台电脑的私人秘钥，不能给别人看的，id_rsa.pub是公共秘钥，可以随便给别人看。把这个公钥放在GitHub上，这样当你链接GitHub自己的账户时，它就会根据公钥匹配你的私钥，当能够相互匹配时，才能够顺利的通过git上传你的文件到GitHub上。

把pub的公共密钥放在GitHub上不用配置，而是在GitHub的setting中，找到SSH keys的设置选项，点击New SSH key把你的id_rsa.pub里面的信息复制进去。（这步不要漏了）
